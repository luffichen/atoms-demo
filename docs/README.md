## 文档规则

- 原子任务文档只描述用户结果、行为边界与验收标准，不包含具体技术实现。
- 每个任务只有一个主要用户结果，并包含独立性与原子性检查。
- 任务间的建议实现顺序只记录在索引中，不把多个任务重新捆绑成大需求。
- 飞书原始资料保持原样归档；访谈形成的新结论以首版产品范围和原子任务文档为准。
