# 预览已发送图片

## 目标

用户可以在对话中看清已经发送的单张参考图片，并方便返回原阅读位置。

## 需求

1. 已发送消息中的每张图片显示可识别缩略图。
2. 点击缩略图打开单图大图预览。
3. 关闭预览后返回原对话位置。
4. 首版不提供下载、编辑、旋转或多图轮播。

## 验收标准

- 点击任意已发送图片都能查看对应大图。
- 关闭后对话滚动位置不跳动。
- 多图消息逐张点击时显示正确图片。
- 页面中不存在未纳入首版的图片操作。

## 不在本任务范围内

- 发送前图片预览。
- 图片上传和校验。
- 图片下载。

## 独立性与原子性检查

- **单一职责**：只负责查看一张已经发送的图片。
- **可独立验证**：使用含多图的历史消息即可验证。
- **结论**：任务足够独立且原子，无需继续拆分。
