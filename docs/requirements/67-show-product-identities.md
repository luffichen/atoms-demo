# 展示产品与参与者身份

## 目标

用户能清楚识别产品、当前游客和智能体，同时不复制参考产品的商标素材。

## 需求

1. 品牌区显示 `Atoms Demo` 和自有简洁图标。
2. 不使用 Atoms 官方 Logo 或专有素材。
3. 游客头像使用名称首字符生成，不允许上传。
4. 同一游客头像在所有页面保持一致，`default` 显示 `D`。
5. 智能体固定显示为 `luffi · 工程师`，使用固定自有头像。
6. 不按项目创建多个智能体身份。

## 验收标准

- 各页面品牌名称和图标一致且不使用官方 Atoms 商标。
- 多个游客头像按名称产生并跨页面一致。
- 所有项目中的智能体名称、角色和头像一致。

## 不在本任务范围内

- 上传或编辑头像。
- 多智能体角色。
- 深色主题。

## 独立性与原子性检查

- **单一职责**：只定义产品和对话参与者的固定身份呈现。
- **可独立验证**：使用多个游客和项目对比即可验证。
- **结论**：任务足够独立且原子，无需继续拆分。
