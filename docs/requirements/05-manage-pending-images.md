# 管理待发送图片

## 目标

用户发送消息前能够看清已添加的图片，并撤销不想发送的单张图片。

## 需求

1. 每张待发送图片都以缩略图展示在输入区。
2. 文件选择和剪贴板粘贴的图片采用相同展示方式。
3. 用户可以单独移除任意一张待发送图片。
4. 移除图片不会清空文字或其他待发送图片。

## 验收标准

- 添加多张图片后，每张图片均有可识别的缩略图。
- 用户可以移除其中任意一张，其余图片顺序和文字保持不变。
- 文件选择和粘贴得到的图片均可预览和移除。

## 不在本任务范围内

- 添加或校验图片。
- 放大查看图片。
- 调整图片顺序。
- 编辑或裁剪图片。
- 发送消息。

## 独立性与原子性检查

- **单一职责**：只管理发送前已经加入的图片。
- **可独立验证**：可使用预置的待发送图片验证展示和单张移除。
- **结论**：任务足够独立且原子，无需继续拆分。
