# 使用主导航

## 目标

用户可以从统一侧边栏在首页和“我的项目”之间切换，并随时确认当前游客。

## 需求

1. 展开侧边栏显示产品名称 `Atoms Demo`、当前游客、首页和“我的项目”。
2. 当前所在页面具有明确选中状态。
3. 点击首页进入当前游客首页。
4. 点击“我的项目”进入当前游客项目列表。
5. 首页和“我的项目”的游客入口可切换游客。
6. 项目对话页的游客名称仅用于展示，不可打开切换列表。
7. 首版不展示“资源”或其他导航入口。

## 验收标准

- 首页与“我的项目”可相互跳转，当前项始终正确高亮。
- 两个页面均显示当前游客。
- 从项目页不能通过游客名称切换身份。
- 导航中只有已确认的入口，不出现“资源”等额外功能。

## 不在本任务范围内

- 侧边栏折叠。
- 窄屏抽屉。
- 游客列表内容和切换失败。
- 各页面主内容。

## 独立性与原子性检查

- **单一职责**：只负责一级页面导航。
- **可独立验证**：点击两个入口并检查选中状态与页面即可验证。
- **结论**：任务足够独立且原子，无需继续拆分。
