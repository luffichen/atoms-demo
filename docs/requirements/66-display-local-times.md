# 显示本地时间

## 目标

项目、消息和事件时间以用户熟悉的本地时间一致展示。

## 需求

1. 使用桌面 Chrome 的本地时区。
2. 更早日期显示为 `YYYY/MM/DD`。
3. 当天内容显示为 `HH:mm`。
4. 悬停时间可查看完整本地日期和时间。
5. 项目卡片、消息和事件采用同一规则。

## 验收标准

- 改变系统时区后重新打开页面，时间按新本地时区显示。
- 当天和更早内容分别采用约定格式。
- 悬停可看到完整时间。
- 同一事件在不同页面的时间含义一致。

## 不在本任务范围内

- 对话日期分隔线。
- 修改项目最近更新时间。
- 用户选择时区。

## 独立性与原子性检查

- **单一职责**：只格式化用户可见时间。
- **可独立验证**：使用不同日期和系统时区即可验证。
- **结论**：任务足够独立且原子，无需继续拆分。
