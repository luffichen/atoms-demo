# 导航应用预览

## 目标

用户可以在生成应用中浏览不同页面，并便捷地刷新或在独立标签页查看。

## 适用范围

- 仅适用于可在浏览器中运行的网页项目。

## 需求

1. 应用查看器提供后退、前进、刷新和“在新标签页打开”。
2. 查看器显示当前相对路径，但首版不可编辑。
3. 项目内部链接在应用查看器内导航。
4. 外部网站链接始终在新的浏览器标签页打开。
5. 外部链接不得替换当前应用预览。

## 验收标准

- 内部多页面导航后，后退和前进可正确使用。
- 刷新后仍显示当前项目页面。
- 新标签页打开当前预览位置。
- 相对路径随内部导航更新但不能输入修改。
- 外部链接打开新标签页，原预览保持项目页面。

## 不在本任务范围内

- 预览启动或失败恢复。
- 编辑地址。
- 生成应用自身功能正确性。

## 独立性与原子性检查

- **单一职责**：只提供应用预览中的浏览操作。
- **可独立验证**：使用含内部与外部链接的应用即可验证。
- **结论**：任务足够独立且原子，无需继续拆分。
