# 浏览项目文件树

## 目标

用户可以从“文件”模式浏览项目拥有的源文件和资源，不接触项目外或系统管理内容。

## 需求

1. “文件”模式左侧展示可展开和收起的文件树。
2. 展示项目拥有的源文件、配置文件和静态资源。
3. 隐藏依赖缓存、构建产物、运行时临时文件和系统内部元数据。
4. 不允许浏览当前项目之外的文件。
5. 选择文件后，在同一模式右侧显示内容。
6. 首版不提供文件名搜索。
7. 首次进入时只展开项目根目录，其他目录默认收起。
8. 点击文件事件进入时，自动展开到目标文件并选中。
9. 首版不记忆上次目录展开状态。
10. 直接进入模式时，根目录存在 README 则自动打开；否则内容区显示“请选择文件”。

## 验收标准

- 可以展开目录并选择项目源文件。
- 约定隐藏内容不出现在树中。
- 无法通过树或路径操作访问项目外文件。
- 选择不同文件时内容区正确切换。

## 不在本任务范围内

- 文件内容的具体呈现。
- 文件状态标记。
- 修改文件。

## 独立性与原子性检查

- **单一职责**：只负责安全地列出和选择项目文件。
- **可独立验证**：使用包含可见与隐藏类别的项目即可验证。
- **结论**：任务足够独立且原子，无需继续拆分。
