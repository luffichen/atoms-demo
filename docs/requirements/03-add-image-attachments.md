# 添加图片附件

## 目标

用户可以在文字需求中加入参考图片，为智能体提供页面风格或内容方面的补充上下文。

## 需求

1. 用户可以通过输入区的附件入口选择本地图片。
2. 用户可以直接在输入区粘贴剪贴板中的图片。
3. 首页首次需求和项目内后续消息均支持这两种添加方式。
4. 成功添加的图片进入当前消息的待发送图片列表。
5. 添加图片不会改变或清空已经输入的文字。

## 验收标准

- 从附件入口选择一张符合要求的图片后，该图片出现在当前消息中。
- 在输入区粘贴一张符合要求的剪贴板图片后，该图片出现在当前消息中。
- 添加图片前已经输入的文字保持不变。
- 首页和项目对话页均能完成上述操作。

## 不在本任务范围内

- 判断图片类型、大小或数量是否合规。
- 展示缩略图或移除图片。
- 发送消息。
- 处理图片添加或发送失败。

## 独立性与原子性检查

- **单一职责**：只负责把用户选择或粘贴的图片加入当前消息。
- **可独立验证**：分别通过文件选择和剪贴板粘贴即可验证。
- **结论**：任务足够独立且原子，无需继续拆分。
