# 从项目列表打开项目

## 目标

用户可以从“我的项目”直接进入所选项目，查看其持续对话和生成结果。

## 需求

1. 整张项目卡片均可点击。
2. 点击卡片后打开该项目的对话页面。
3. 打开的项目必须属于当前游客。
4. 打开项目后恢复该项目已有的对话和生成结果。
5. 首版不要求卡片提供其他操作入口。

## 验收标准

- 点击卡片缩略图、名称或其他空白区域均打开同一个项目。
- 打开后看到与所选卡片一致的项目名称和历史内容。
- 不会因同名项目而打开错误项目。
- 项目页显示当前游客且不允许切换游客。

## 不在本任务范围内

- 项目列表加载和排序。
- 项目页内部的具体对话、文件和预览展示。
- 项目不存在或不可用时的错误恢复。

## 独立性与原子性检查

- **单一职责**：只负责从一张卡片进入对应项目。
- **可独立验证**：使用两个项目及同名项目即可验证准确导航。
- **结论**：任务足够独立且原子，无需继续拆分。
